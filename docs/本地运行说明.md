# 智批 AI 本地运行说明

## 前置环境

- Node.js 22 或更高版本；
- npm；
- 如需运行独立后端：Python 3.11；
- 如需运行 MySQL、Redis 完整环境：Docker Desktop。

## 运行前端比赛 Demo

进入包含 `package.json` 的完整源码目录：

```powershell
cd "你的智批AI完整源码目录"
npm.cmd install
npm.cmd run dev
```

打开终端显示的本地地址，通常是：

```text
http://localhost:3000
```

若 3000 端口被占用，请使用终端提示的其他端口，例如 `http://localhost:3001`。

前端比赛 Demo 已包含稳定的站内演示接口，不配置 OCR 或大模型密钥也能演示数学批改、作文评分和学情分析。

## 运行 FastAPI 后端

```powershell
cd "你的智批AI完整源码目录\backend"
py -3.11 -m venv .venv
.\.venv\Scripts\python.exe -m pip install -r requirements.txt
.\.venv\Scripts\python.exe -m uvicorn app.main:app --reload --port 8000
```

启动后打开：

```text
http://localhost:8000/docs
```

可以在该页面测试 OCR 适配、数学分步评分、作文四维评分和学生学情接口。

## Docker 完整环境

```powershell
cd "你的智批AI完整源码目录"
Copy-Item backend\.env.example backend\.env
docker compose up --build
```

默认端口：

- FastAPI：8000
- MySQL：3306
- Redis：6379

## 常见问题

### PowerShell 提示无法运行 npm.ps1

使用 `npm.cmd`，不要使用 `npm`：

```powershell
npm.cmd install
npm.cmd run dev
```

### 页面端口不是 3000

以终端输出的 `Local` 地址为准。

### 如何停止

在运行服务的终端中按 `Ctrl + C`。
